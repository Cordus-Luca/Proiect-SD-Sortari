# Proiect-SD-Sortari
Am abreviat numele sortarilor si a tipurilor de teste pentru utilizare mai rapida.

Tip Test:
ASC = Aproape sortat crescator
ASD = Aproape sortat descrescator
R = Random
C = Constant

Algoritm sortare:
B = BubbleSort
C = CountSort
R = RadixSort
M = MergeSort
Q = QuickSort
L = Algoritmul implementat de limbaj
